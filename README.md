# Data Warehouse and Analytics Project

Welcome to the **Data Warehouse and Analytics Project** repository! 🚀  
This project demonstrates a complete data warehousing and analytics solution, from building a modern SQL data warehouse to generating actionable business insights. Designed as a portfolio project, it highlights industry best practices in data engineering and analytics.

---

# Project Requirements

## Building the Data Warehouse (Data Engineering)

### Objective
Develop a modern data warehouse using SQL Server to consolidate and analyze business data for reporting and decision-making.

### Specifications
- **Data Sources:** Import data from multiple source systems provided as CSV files.
- **Data Cleaning:** Handle missing values, duplicates, and data quality issues before analysis.
- **Data Integration:** Combine datasets into a centralized analytical model.
- **Scope:** Focus only on the latest snapshot of data (historization not required).
- **Documentation:** Create clear documentation for tables, relationships, and transformations.

---

## Analytics & Reporting (Data Analytics)

### Objective
Use SQL queries to generate business insights and support strategic decisions.

### 📈 Key Analytics Areas
- Customer behavior analysis
- Product performance analysis
- Sales trend analysis
- Revenue and profit insights
- Regional performance comparison

These insights help stakeholders understand business performance and improve operations.

---

# Data Architecture

The project follows the **Medallion Architecture** approach:

1. **Bronze Layer** – Raw data ingestion from source systems  
2. **Silver Layer** – Cleaned and transformed data  
3. **Gold Layer** – Business-ready analytical tables for reporting
